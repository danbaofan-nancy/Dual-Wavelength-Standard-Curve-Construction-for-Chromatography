# 双波长法凝胶层析数据分析

## 项目简介

本项目用于处理 **Sephadex G75 凝胶层析** 分离细胞色素 C（Cyt c）与核黄素磷酸钠（FMN）的实验数据。  
通过双波长分光光度法（408 nm / 444 nm）计算混合洗脱液中两组分的浓度，并绘制洗脱曲线。

**改进点**：
- 96 孔板替代小指管收集洗脱液
- 双波长法替代目测法进行定量
- **解决了传统方法无法区分混合液各组分浓度的问题**

## 文件说明

| 文件 | 说明 |
| :--- | :--- |
| `吸光度数据.xlsx` | 原始吸光度数据（A408 / A444） |
| `绘制洗脱曲线.py` | 主程序：计算浓度、保存结果、绘制洗脱曲线 |
| `计算结果.xlsx` | 输出：各组分浓度（mol/L 及 mg/mL） |
| `洗脱曲线.png` | 输出：洗脱曲线图 |
